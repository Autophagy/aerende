Note Creation/Editing/Deleting
------------------------------

====== ===========================
Key    Action
====== ===========================
``n``  Create a new note
``e``  Edit the focused note
``d``  Delete the focused note
``+``  Increment a note's priority
``-``  Decrement a note's priority
